# Требования к разработке модели машинного обучения

## Основные задачи
1. Создать признак для разметки данных
2. Провести классификацию клиентов

## Рекомендации по реализации
- Возможно ранжирование клиентов по степени лояльности на основе:
  - Суммы покупок
  - Баланса накоплений
- Ограничение анализа: данные только за 1 месяц

## Ожидаемый результат
Модель классификации клиентов по уровню лояльности с использованием финансовых показателей из датасета.
